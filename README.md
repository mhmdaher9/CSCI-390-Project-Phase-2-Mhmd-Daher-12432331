# Project Overview
F1 For Newbies is a React-based educational website designed to introduce beginners to the world of Formula 1 racing. The website provides users with basic Formula 1 knowledge, team information, race schedules, multimedia content, and interactive animations in a beginner-friendly format.

This project was developed as part of the CSCI-390 Phase 2 project requirements.

---

# Features

- Interactive homepage with Formula 1 themed design
- Animated starting lights sequence
- Audio effects synchronized with lights animation
- Beginner’s guide to Formula 1
- Manufacturers/Teams page
- 2026 Formula 1 race calendar
- Embedded educational video
- Responsive design for desktop and mobile devices
- React component-based architecture
- Online deployment using Vercel
